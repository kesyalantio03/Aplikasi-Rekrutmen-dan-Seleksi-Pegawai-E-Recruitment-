# Aplikasi-Rekrutmen-dan-Seleksi-Pegawai-E-Recruitment-
Aplikasi E-Rekrutmen Pegawai untuk PT. Siantar Codes Academy. Sistem web untuk pendaftaran kandidat, pengelolaan profil, upload dokumen (CV, KTP, Ijazah, foto), melihat lowongan, tata cara, dan pengumuman. Dibangun dengan UI modern dan database MySQL untuk proses rekrutmen yang lebih efisien.
